formatter

Morgan log formatter.

Morgan log formatter. Returns an object containing the desired request and response attributes. The expected output should be a JSON string as bunyan will take the string and merge that JSON with the output (which is the desired result). The expected format is: { remote-addr: , date: , method: , url: , HTTP: , user-agent: , referrer: , status: , res[content-length]: , response-time: ms, }
